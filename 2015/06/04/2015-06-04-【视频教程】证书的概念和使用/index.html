【视频教程】证书的概念和使用

  • Java
  • 密码学
  • 极客学院

课程背景

无论是公钥密码还是数字签名，其中公钥都扮演了重要的角色。然而，如果不能判断自己手上的公钥是否合法，就有可能遭到中间人攻击。为了应对此种情况，就需要使用证书技术对公钥的合法性提供证明。

核心内容

  • 证书的概念和应用场景
  • 公钥基础设施的概念和作用
  • 使用 VeriSign 服务生成证书

在此只提供视频第一课，具体课程请至极客学院学习~
