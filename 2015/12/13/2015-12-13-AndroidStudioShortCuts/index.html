大多数人都不知道的Android Studio快捷键

  • android

当然网上也有很多关于 Android Studio 快捷键的文章，但感觉太冗杂，而且不同的平台快捷键又不一样，导致对着 Mac 打 Win#Eclipse 的快捷键不起作用。所以本篇文章整理的目的在于，列出使用频繁的快捷键并告诉你在哪里设置它们，而且文字看不懂的还有动画，这样即使你用的键盘跟我不一样，你也知道应该去哪里设置相应的键位。

全局说明：

  • 我用的是 Mac，键位是 Mac OS X 10.5+，如下图

  • 若你要修改对应的快捷键，去 Settings->Keymap 中修改，如下图

0x01 定位目标

功能 快捷键 位置
搜索类(Enter class name) Cmd+O Main menu->Navigate->Class
搜索方法或字段(Enter symbol name) Cmd+Option+O Main menu->Navigate->Symbol
搜索文件、目录(Enter file name) Cmd+Shift+O Main menu->Navigate->File
Search Everywhere Shift+Shift
查看字段、方法、类、文件都在哪些地方被使用了(全局) Option+F7 Main menu->Edit->Find->Usages
查看字段、方法、类、文件都在哪些地方被使用了(当前文件) Cmd+F7 Main menu->Edit->Find->Find Usages in File
搜索任意快捷键(Enter action or option name) 很强大！知道快捷操作关键字即可！ Cmd+Shift+A Main menu->Help->Find Action

0x02 查看结构

功能 快捷键 位置
查看定义 Cmd+B Main menu->Navigate->Declaration
智能选取 Option+Up/Down Editor Actions->Extend Selection/Shrink Selection
查看本类结构 Cmd+F12 Cmd+7 Main menu->Navigate->File Structure Other->Structure
查找和替换 Cmd+F Cmd+R Cmd+Shift+F Cmd+Shift+R Main menu->Edit->Find/Replace
查看doc F1 Ctrl+J Main menu->View->Quick Documentation

0x03 视图切换

功能 快捷键 位置
显示目录窗口 Cmd+1 Other->Project/Home Directory
显示收藏窗口 Cmd+2 Other->Favorites
显示AndroidMonitor Cmd+6 Other->Android Monitor
显示类结构 Cmd+7 Other->Structure
最近查看文件窗口 Cmd+E Main menu->View->Recent Files
最近修改文件窗口 Cmd+Shift+E Main menu->View->Recent Changed Files
在当前编辑器切换标签 Cmd+Shift+[/] Main menu->Window->Editor Tabs->Select Previous Tab/Select Next Tab
切换到顶部导航栏 Cmd+Up/Down Main menu->Navigate->Jump to Navigation Bar/Jump to Source
视图切换Switcher Ctrl+Tab Other->Switcher

0x04 编辑代码

功能 快捷键 位置
LiveTemplate Cmd+J Main menu->Code->Insert Live Template
Generate Cmd+N Main menu->Code->Generate
智能提示 Option+Enter Other->Show Intention Actions
关闭当前活动窗口 Cmd+W Main menu->Window->Editor Tabs->Close
格式化代码 Cmd+Option+L Main menu->Code->Reformat Code
将当前代码段上下移动 Cmd+Shift+Up/Down Main menu->Code->Move Statement Up/Move Statement Down
将当前行上下移动 Shift+Option+Up/Down Main menu->Code->Move Line Up/Move Line Down
插入多个光标协同编辑 Shift+Option+左键
从当前位置剪切到末尾 Ctrl+K Editor Actions->Cut up to Line End

0x05 重构

功能 快捷键 位置
移动文件 F6 Main menu->Refactor->Move
拷贝文件 F5 Main menu->Refactor->Copy
重命名 Shift+F6 Main menu->Refactor->Rename
修改方法签名 Cmd+F6 Main menu->Refactor->Change Signature
修改参数类型 Cmd+Shift+F6 Main menu->Refactor->Type Migration
引入一个局部变量 Cmd+Shift+V
引入一个参数 Cmd+Shift+P
引入一个类变量 Cmd+Shift+F
引入一个方法 Cmd+Shift+M
引入一个常量 Cmd+Shift+C
显示重构菜单 Ctrl+T Main menu->Refactor->Refactor This

0x06 查看代码

功能 快捷键 位置
查询某方法参数信息 Cmd+P Main menu->View->Parameter Info
跳到方法定义 Cmd+B Main menu->Navigate->Declaration
跳到变量的定义类 Cmd+Shift+B Main menu->Navigate->Type Declaration
跳到方法在父类或接口的定义处 Cmd+U Main menu->Navigate->Super Method
跳到上/下一方法 Ctrl+Up/Down Main menu->Navigate->Previous Method/Next Method
跳到上/下一次光标查看处(阅读源代码时非常有用) Cmd+Option+Left/Right Main menu->Navigate->Back/Forward
跳到指定行 Cmd+L Main menu->Navigate->Line
显示方法层级结构 Cmd+Shift+H Main menu->Navigate->Method Hierarchy
调用层级结构 Cmd+Option+H Main menu->Navigate->Imports Hierarchy

0x07 Run & Debug

功能 快捷键 位置
Run Ctrl+R Main menu->Run->Run
build Cmd+F9
Debug Ctrl+D Main menu->Run->Debug
单步进入 F7 Main menu->Run->Step Into
单步跳过 F8 Main menu->Run->Step Over
跳过本次debug F9 有待商榷
debug时执行选中的语句 Option+F8 有待商榷
要做挖掘机而非推土机
